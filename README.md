# 📱 Manga Reader App

A modern Android Manga Reader application built with **Kotlin** and **XML**, designed to provide a smooth, clean, and enjoyable reading experience.

---

## ✨ Features

* 📚 Browse manga content easily
* 📖 Smooth reading experience
* 🎨 Clean and modern UI design
* 🌗 Gradient overlays for better visuals
* ⚡ Fast and lightweight performance

---

## 🛠️ Tech Stack

* Kotlin (Android Development)
* XML (UI Design)
* Android Studio

## 🚀 Getting Started

### Prerequisites

* Android Studio installed
* Minimum SDK supported

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Mennaouiaymen66/manga-reader-app.git
   ```
2. Open the project in **Android Studio**
3. Let Gradle sync finish
4. Run the app on an emulator or physical device
